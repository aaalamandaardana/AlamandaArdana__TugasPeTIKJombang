Ada alasan emosional di balik tangis pemain asal Brasil itu usai melakukan debut gemilangnya di Liga Champions. Richarlison menjelaskan alasan mengapa dirinya tak kuasa menahan air mata setelah mencetak dua gol untuk Tottenham Hotspur dalam kemenangan 2-0 atas Marseille di Liga Champions. Penyerang asal Brasil, yang bergabung dengan Spurs dari Everton seharga Â£50 juta pada musim panas ini, membuka rekening golnya untuk klub di kompetisi Eropa, Kamis (8/9) dini hari WIB. Keluarganya hadir untuk menyaksikan peristiwa yang tak terlupakan bagi pemain berusia 25 tahun itu, yang menghasilkan suasana mengharukan darinya selepas pertandingan.


Mengapa Richarlison menangis usai cetak gol di Liga Champions?

Richarlison mengatakan alasan dirinya menangis selepas peluit akhir dibunyikan: "Ayah saya ada di sini, ia telah bersama saya sepanjang karier saya, ia banyak membantu saya untuk mewujudkan mimpi. Kehadirannya di sini sangat emosional bagi saya hari ini karena ia adalah bagian dari semua kesuksesan saya."

"Setelah mendengar lagunya [Liga Champions] saya mulai tersenyum. Itu adalah impian saya sebagai seorang anak kecil. Berada di sana, mendengarkan lagunya, memainkan pertandingan, saya menjadi sangat emosional dan saya serasa mendapat hadiah dengan mencetak dua gol ini."


Debut cemerlang di Liga Champions

Richarlison melakukan debutnya di Liga Champions melawan Marseille dan membantu Spurs memulai kampanye penyisihan grup mereka dengan tiga poin ketika mencetak dua gol di babak kedua melawan tim Ligue 1 tersebut. Ia menghadapi persaingan sengit untuk mendapatkan tempat di skuad Antonio Conte, bersaing dengan nama-nama beken seperti Harry Kane, Heung-min Son dan Dejan Kulusevski di lini depan.
