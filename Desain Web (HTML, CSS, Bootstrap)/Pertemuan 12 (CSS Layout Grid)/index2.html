Hasil Barcelona vs Viktoria Plzen: Skor 5-1

Barcelona sukses menghajar Viktoria Plzen dengan skor 5-1 dalam partai matchday 1 fase grup Liga Champions 2022/2023 yang digelar di Camp Nou, Kamis (8/9/2022) dini hari WIB. Robert Lewandowski menjadi bintang kemenangan Barcelona dengan membukukan hat-trick alias tiga gol. Dua gol lainnya masing-masing dicetak Franck Kessie dan Ferran Torres. Plzen sempat mencetak gol hiburan lewat Jan Sykora. Berkat hasil ini, Barcelona untuk sementara berhak memuncak klasemen Grup C dengan poin 4. Sementara itu, Plzen terpuruk di dasar klasemen.


Babak Pertama


Barcelona mencoba langsung tampil agresif sejak menit pertama. Hasilnya, Kessie sukses membuka keunggulan memanfaatkan sebuah situasi sepak pojok. Ia meneruskan assist Jules Kounde. Barcelona baru bisa menggandakan keunggulan pada menit ke-34 lewat aksi Lewandowski. Menerima umpan Sergi Roberto, Lewy melakukan kontrol sebelum melepas tembakan melengkung untuk menaklukkan Jindrich Stanek. Menit ke-44, Plzen memperkecil ketinggalan lewat sundulan Sykora. Namun, sesaat sebelum turun minum, Lewandowski mencetak gol keduanya untuk membawa Barca memimpin 3-1.
